## Sobre o Projeto

O ToDoList é uma aplicação full-stack que combina:

- **Backend**: API REST em ASP.NET Core com .NET 9
- **Frontend**: Aplicação web em Angular 20
- **Banco de Dados**: PostgreSQL
- **Autenticação**: JWT com ASP.NET Core Identity
